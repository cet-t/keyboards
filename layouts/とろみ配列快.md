# とろみ配列快

四つ角はできる限り使わない

## 配列図

|     | ら  | か  | ん  |     |     | ゆ  | う  | よ  |     |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| は  | さ  | な  | た  | ゜  | ー  | え  | い  | お  | ゛  |
|     |     | わ  | ま  |     |     | あ  | や  |     |     |

あ  : あ(m)
き  : か+い(e+k)
だ  : た+゛(f+;)
ぱ  : は+゜(a+f)
じゃ: さ+や+゛(s+,+;)
ぴょ: は+よ+゜(a+o+;)
